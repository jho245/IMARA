# SEO-Checkliste

## On-Page

- [ ] Meta-Description für jede Seite (155–160 Zeichen)
- [ ] Title-Tags (50–60 Zeichen, Keyword + Firmenname)
- [ ] Eine H1 pro Seite
- [ ] H2–H6 Hierarchie einhalten
- [ ] Keyword-Recherche abgeschlossen
- [ ] Interne Verlinkung zwischen Seiten
- [ ] Schema Markup eingebunden (LocalBusiness, Service)
- [ ] Canonical URLs gesetzt
- [ ] Alt-Texte für alle Bilder

## Technisch

- [ ] sitemap.xml hochgeladen und aktuell
- [ ] robots.txt konfiguriert
- [ ] SSL-Zertifikat aktiv (HTTPS)
- [ ] Ladezeit < 3 Sekunden
- [ ] Lighthouse Score > 90
- [ ] Mobile-freundlich (Google Mobile-Friendly Test)
- [ ] Strukturierte Daten validiert (Google Rich Results Test)

## Off-Page

- [ ] Google My Business Eintrag erstellt und verifiziert
- [ ] In relevante Branchenverzeichnisse eingetragen
- [ ] Google Search Console eingerichtet
- [ ] Sitemap in Search Console eingereicht
